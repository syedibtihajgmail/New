Know Your Lookup

This suite offers a wide range of network tools. Here's what each tab helps you do:

🌐 IP Lookup

Find the geographical location and ISP of any IP address, or look up your own.

🔍 DNS Lookup

Retrieve A, AAAA, TXT, NS, and CNAME records for a domain.

📧 MX Lookup

Show mail exchange records and resolved IPs for email troubleshooting.

🛡️ DMARC Suite

Lookup and generate DMARC policies to protect against email spoofing.

📨 SPF Suite

Check or build SPF records to define authorised mail senders.

🔑 DKIM

Search for DKIM keys using common selector names.

↪️ CNAME

View canonical (alias) records for any hostname.

🌍 AAAA

Lookup IPv6 addresses for modern connectivity checks.

🕵️ IP Leak Test

Reveal your public IP to verify VPN/proxy effectiveness.

🔒 Security Checks

Check IP reputation (VPN/Proxy/Spamhaus) and domain SSL/DNS health.

⚡ Speed Test

Measure ping, download, and upload speeds with server and geo info.
